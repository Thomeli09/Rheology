# Rheology
## Objectifs:
Le but de cette page est de créer une application capable de résoudre la plupart des problèmes rencontrés dans le monde de l'ingénieur des constructions et plus particulièrement les soucis liés à la rhéologie.
## Note:
- Le langage de programmation utilisé est principalement Python, mais peut changer éventuellement.
